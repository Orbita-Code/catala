# Illustration Prompts for Aprenc Catala

## Style Guide
**Consistent prefix for ALL prompts:**
> 3D cartoon illustration, cute Pixar-style, for a children's language learning app (ages 5-8). Simple, clear, colorful, white background. Single object/character centered. PNG 512x512.

**Naming convention:** `public/illustrations/{theme}/{word}.png`
Example: `public/illustrations/la-classe/llapis.png`
